Accessibility Messenger

Bringing everyone together

Myo Integration

Using the Myo SDK, we were able to develop a basic messaging web app using sign language detected by the armband.

Accessible

All types of people no matter what their disabilities are, can access this app and use it to its full potential. There is no advantage to non-disabled people, making it great for disabled people to use.

Simple UI

The simple web app is very easy and straightforward to use. No need to learn complicated things to be able to use it.
